Tor Traffic Analysis Research (MSc)

Shadow Simulator • Network Security Research • Ethics • Reporting

Objective (UK relevance)

Study feasibility and limitations of deanonymisation-style traffic analysis in Tor and explain why real-world exploitation is constrained—useful for threat modelling, research, and security assessment roles.

Tools & Methods

  • Shadow simulator controlled environment
  • Website fingerprinting + flow correlation concepts
  • Network analysis thinking: noise, timing, routing diversity
  • Clear documentation + ethical analysis

Key findings (what a SOC/Analyst cares about)

  • High “fidelity gap” between simulations and real Tor behaviour.
  • Scale/infrastructure constraints make large experiments hard to reproduce.
  • Ethical restrictions limit aggressive real-network data collection.
  • Practical barriers reinforce Tor’s anonymity in real conditions.

Security skills demonstrated

  • Threat modelling + limitations analysis
  • Evidence-driven reporting
  • Network traffic reasoning
  • Research discipline (great for UK grad roles)
