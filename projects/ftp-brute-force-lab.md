# FTP Brute Force Attack Lab

## Objective
Demonstrate how brute force attacks target FTP services and how to mitigate them.

## Tools Used
- Kali Linux
- Hydra
- Wireshark

## Steps
1. Setup FTP server
2. Performed brute force using Hydra
3. Captured traffic using Wireshark
4. Identified vulnerabilities

## Security Lessons
Weak passwords make FTP vulnerable to brute force attacks.

## Mitigation
- Use strong passwords
- Disable anonymous login
- Implement fail2ban
