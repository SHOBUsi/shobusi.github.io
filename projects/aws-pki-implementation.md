# Two-Tier PKI Design & Implementation (AWS)

## Objective
Design a secure certificate infrastructure to enable encrypted communication.

## Infrastructure
- AWS EC2 instances
- Windows Server 2019
- Root CA & Issuing CA

## Implementation
- Configured certificate lifecycle management
- Enabled TLS on web server
- Verified encryption via packet capture

## Security Impact
Ensures secure communications and prevents man-in-the-middle attacks.

## Skills Gained
- Cloud security architecture
- Certificate services
- Network encryption
